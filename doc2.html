Graphical User Interface (GUI)

As of this moment (12/21/15) the GUI consist of multiple .ui files inside the gtk directory. These .ui files are loaded into variables within the builder class in AutoGentoo.py. All signals sent out by GTK are handled by the dictionaries defined around the end of the AutoGentoo.py file.

Emerge

Package

A package is a program that will be compiled and then installed with emerge. A Gentoo package is defined by a .ebuild file usually located in the portage tree (/usr/portage). The Package class will take a line generated by emerge --pretend and given to by the 'Emerge' class and will generate needed information to compile and install the package.
